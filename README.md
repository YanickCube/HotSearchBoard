# 热搜榜 (Hot Search Board)

一个基于 Vue 3 + Vite 开发的实时热搜榜单展示应用，集成了 AI 智能分析功能。

## 功能特点

- 🔥 实时展示抖音热搜榜单数据
- 🤖 集成 AI 智能分析，为热搜话题提供深度解读
- 🎨 优雅的 UI 设计，流畅的动画效果
- 📱 响应式布局，支持多端适配
- ⚡️ 基于 Vite 构建，开发体验极致

## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- Vite - 下一代前端构建工具
- Element Plus - 基于 Vue 3 的组件库
- Axios - HTTP 请求库
- Silicon Flow API - AI 大模型服务
- 天行数据 API - 热搜数据服务

## 开发指南

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

### 快速开始

```bash

# 进入项目目录
cd hot-search-board 

# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
